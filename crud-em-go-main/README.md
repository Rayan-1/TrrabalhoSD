# CRUD em Go

Este repositório contém uma aplicação simples de CRUD (Create, Read, Update, Delete) desenvolvida em Go. O projeto demonstra a criação de uma API RESTful que permite gerenciar dados de forma eficiente.

## Tecnologias Utilizadas

- **Go**: Linguagem de programação principal.
- **Gin**: Framework web para Go.
- **Gorm**: ORM para interação com o banco de dados.
- **SQLite**: Banco de dados utilizado para persistência de dados.

## Funcionalidades

- Adicionar novos registros.
- Listar todos os registros.
- Atualizar registros existentes.
- Deletar registros.
